# Lab7
## Repository Overview

This repository contains all files related to Lab 7 - Logistic Regression.

### Files Included

* **Lab7.ipynb**
  Main notebook that implements Logistic Regression using the Heart Disease dataset. It includes data analysis, visualization, model training, and evaluation.

* **02_logistic_regression_assignment.ipynb**
  Assignment notebook containing additional exercises and applications of Logistic Regression.

* **heart.csv**
  Dataset used for predicting heart disease.

* **advertising.csv**
  Additional dataset used for practicing classification tasks.

* **README.md**
  Documentation file that explains the project, workflow, and results.

### Summary

The repository demonstrates the full machine learning workflow:

* Data exploration
* Data preprocessing
* Model building
* Model evaluation

All work is implemented using Python and standard data science libraries.
